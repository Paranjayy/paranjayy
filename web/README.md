# Ideaverse Web Hub

This is the high-fidelity frontend for the Ideaverse Orchestrator. It visualizes the data gathered by the Python Intelligence Engine.

## Tech Stack
- **Next.js 14** (App Router)
- **Tailwind CSS** (Glassmorphic Theme)
- **Framer Motion** (Micro-animations)
- **Lucide React** (Icons)

## Getting Started
1. Run the intelligence engine: `python3 ../scripts/repo-dashboard.py`
2. Sync the data: `cp ../PORTFOLIO_DATA.json src/data/portfolio.json`
3. Start the dev server: `npm run dev`
